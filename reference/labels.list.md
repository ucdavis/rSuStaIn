# Extract variable labels from list

Extract variable labels from list

## Usage

``` r
# S3 method for class 'list'
labels(object, ...)
```

## Arguments

- object:

  a [list](https://rdrr.io/r/base/list.html)

- ...:

  unused

## Value

a [vector](https://rdrr.io/r/base/vector.html) of
[character](https://rdrr.io/r/base/character.html) strings corresponding
to `object`
