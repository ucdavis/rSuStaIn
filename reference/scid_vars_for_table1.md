# List SCID variables to display in data summary tables

List SCID variables to display in data summary tables

## Usage

``` r
scid_vars_for_table1()
```

## Value

a [`character()`](https://rdrr.io/r/base/character.html) vector

## Examples

``` r
scid_vars_for_table1()
#> [1] "SCID: mood disorders"          "SCID: substance use disorders"
#> [3] "SCID: anxiety disorders"       "SCID: somatoform disorders"   
```
