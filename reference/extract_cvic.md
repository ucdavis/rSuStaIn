# Extract CVIC element from an object

Extract CVIC element from an object

## Usage

``` r
extract_cvic(object, ...)
```

## Arguments

- object:

  a `"SuStaIn_output"` object (from
  [`run_OSA()`](https://UCD-IDDRC.github.io/fxtas/reference/run_OSA.md))

- ...:

  unused

## Value

a "CVIC" object

## Examples

``` r
if (FALSE) { # \dontrun{
osa_output |> extract_cvic()
} # }
```
