# rSuStaIn: R Wrapper for Python 'pySuStaIn' Package

Provides an R interface to the 'pySuStaIn' Python package
(<https://github.com/ucl-pond/pySuStaIn>) for Subtype and Stage
Inference (SuStaIn). SuStaIn is an unsupervised learning algorithm that
identifies disease subtypes and stages from cross-sectional data. This
package includes wrapper functions for the main SuStaIn algorithms:
Ordinal SuStaIn for ordinal/scored data, Z-score SuStaIn for continuous
biomarkers, and Mixture Model SuStaIn for event-based modeling.
